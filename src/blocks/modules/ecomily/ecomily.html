Партнёрская программа
Партнерская программа состоит из поля для ввода номера сертификата и номера телефона клиента (необязательное поле). После ввода номера сертификата проверяется его уникальность - данный номер еще не должен быть вбит в систему, и он должен существовать. Ошибится с вводом номера допустимо три раза, после этого ввод номера сертификата блокируется на 60 минут. Это сделано для защиты от пользователей, желающих захватить множество сертификатов путем перебора номеров в попытке их угадать
